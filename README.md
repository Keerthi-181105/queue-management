# 🏥 Healthcare Queue Management System

A smart healthcare queue management system that enables patients to join queues, book appointments, and track real-time status, while doctors efficiently manage patient flow and view analytics.

---

## ✨ Features

### 👤 Patient Portal
- Join doctor queue with live position tracking
- Real-time updates and estimated waiting time
- Smart notifications when turn is near
- Book appointments with preferred doctors
- Search doctors by specialty and availability

---

### 🩺 Doctor Dashboard
- Manage patient queue efficiently
- Call next patient in real time
- View patient details instantly
- Track daily performance stats
- Mark appointments as completed

---

### 📊 Analytics Dashboard
- Total patients, wait time, satisfaction metrics
- Hourly patient flow visualization
- Specialty-wise performance comparison
- Top-performing doctors ranking
- Smart system insights

---

### 📅 Appointment System
- Browse doctor directory
- Search and filter doctors
- Schedule appointments with time slots
- Email confirmation support

---
